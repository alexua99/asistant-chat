# eSIM Chat Assistant - WordPress Plugin

Плагин для интеграции eSIM чат-бота в WordPress сайт.

## Установка

1. Скопируйте папку `esim-chat-wp-plugin` в `/wp-content/plugins/` вашего WordPress сайта
2. Активируйте плагин через админ-панель WordPress (Плагины -> Установленные плагины)
3. Перейдите в Настройки -> eSIM Chat
4. Укажите URL вашего сервера eSIM чата (например: `https://your-domain.com` или `http://localhost:3000`)
5. Сохраните настройки

## Использование

### Шорткод

Добавьте шорткод на любую страницу или пост:

```
[esim_chat]
```

С параметрами размера:

```
[esim_chat width="100%" height="600px"]
```

### В шаблоне темы

```php
<?php echo do_shortcode('[esim_chat]'); ?>
```

### В виджете

Добавьте шорткод в текстовый виджет или используйте блок "Шорткод".

## Требования

- WordPress 5.0 или выше
- PHP 7.4 или выше
- Работающий сервер eSIM чата (отдельное приложение)

## Настройки

Плагин имеет следующие настройки:

- **API URL сервера** - URL вашего сервера eSIM чата
- **Включить чат** - переключатель для включения/выключения чата

## Особенности

- ✅ Красивый современный интерфейс чата
- ✅ Поддержка полей для email, номера заказа и ICCID
- ✅ Анимация печати при ответе
- ✅ Адаптивный дизайн для мобильных устройств
- ✅ Сворачивание/разворачивание чата
- ✅ История диалога
- ✅ Легкая настройка через админ-панель

## Разработка

Для разработки и тестирования:

1. Убедитесь, что ваш сервер eSIM чата запущен и доступен
2. Настройте CORS на сервере для разрешения запросов с вашего WordPress домена
3. Используйте шорткод на тестовой странице WordPress

## Поддержка

Если у вас возникли проблемы:

1. Проверьте, что сервер eSIM чата запущен и доступен
2. Проверьте настройки API URL в админ-панели
3. Откройте консоль браузера (F12) для просмотра ошибок
4. Убедитесь, что CORS настроен правильно на сервере
