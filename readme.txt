Интеграция eSIM чат-бота для консультаций по eSIM технологиям, операторам и устройствам.

== Описание ==

Плагин eSIM Chat Assistant позволяет интегрировать умный чат-бот на ваш WordPress сайт. Чат-бот специализируется на консультациях по:

* eSIM технологиям и установке
* Мобильным операторам в разных странах
* Совместимости устройств
* Настройке eSIM на iOS и Android
* Устранению неполадок

== Установка ==

1. Загрузите папку плагина в `/wp-content/plugins/`
2. Активируйте плагин через меню 'Плагины' в WordPress
3. Перейдите в Настройки -> eSIM Chat
4. Укажите URL вашего сервера eSIM чата
5. Используйте шорткод [esim_chat] на любой странице или в виджете

== Использование ==

**Шорткод:**
[esim_chat]

**С параметрами:**
[esim_chat width="100%" height="600px"]

**В шаблоне темы:**
<?php echo do_shortcode('[esim_chat]'); ?>

== Настройки ==

* **API URL сервера** - URL вашего сервера eSIM чата (например: https://your-domain.com или http://localhost:3000)
* **Включить чат** - включить/выключить отображение чата

== Часто задаваемые вопросы ==

= Нужен ли отдельный сервер? =

Да, вам нужен отдельный сервер с развернутым приложением eSIM чата. Этот плагин только отображает интерфейс чата и подключается к вашему серверу.

= Как настроить CORS? =

Убедитесь, что ваш сервер eSIM чата настроен для работы с вашим WordPress доменом. В настройках сервера должны быть разрешены запросы с вашего домена.
